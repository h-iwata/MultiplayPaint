Overview

This is the documentation and reference for the Exit Games Photon C++ Client API. This Doc ships with the the C++ client library SDKs for the platforms Windows, iOS, OS X, Marmalade, Android NDK, Blackberry 10 NDK, PS3 and Linux. Photon SDKs are also available for various platforms in objective C, C#, Java, JavaScript, Flash and Lua.

Photon is a multiplayer development framework to build real-time multiplayer games and applications for various platforms. It consists of a Server SDK and Client SDKs for the platforms listed above.

Photon provides a low-latency communication-layer for realtime games, based on UDP). This protocol allows reliable and unreliable transfer of data in "commands". On top of this, an operation- and event-framework is established to ease development of your own multiplayer applications.

Server-side, the "Lite" application offers the basic operations that we felt useful for most multiplayer games. On the client side, these operations and events are part of of the API.

The Photon C++ Client consists out of these parts:

Common - data types and utilities
Photon - Photons core networking API + The 'Lite' room logic
LoadBalancing - a high level API on top of Photon, which adds multi-server handling and a lot more
Chat - a high level Chat API
